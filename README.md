# Azure Automation & Scripting

A curated collection of PowerShell, Microsoft Graph API, and Azure Automation scripts for managing Microsoft 365, Entra ID, Azure, and Intune environments.

## 🎯 Purpose

This repository contains practical, production-ready scripts to automate repetitive administrative tasks, improve efficiency, and ensure consistency across Microsoft cloud services.

## 📂 Main Folders

- **Entra-ID** – User management, group automation, licensing, and Conditional Access scripts
- **Microsoft-365** – Exchange Online, Teams, SharePoint, and tenant administration
- **Intune** – Device management, compliance policies, and app deployment
- **Azure** – Resource management, monitoring, and cost optimization scripts
- **Reporting** – Advanced reporting and audit scripts
- **Utilities** – Reusable functions and helper scripts
